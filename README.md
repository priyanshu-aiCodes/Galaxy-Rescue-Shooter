# 🚀 Galaxy Space Shooter

A fast-paced 2D Space Shooter built with HTML5 Canvas and JavaScript.

Created by **Priyanshu Sharma** for an AI Game Development Bootcamp.

---

## 🎮 Features

- 🚀 Smooth spaceship controls
- 🔫 Fast shooting mechanics
- 👾 Multiple enemy variants
- 👹 Epic boss battles
- ❤️ Health system
- 🏆 Score tracking
- 📈 Dynamic level progression
- ⚡ Collectible power-ups
- ⚠ Boss arrival countdown
- 🎵 Immersive background music
- 💥 Particle explosion effects
- ⏸ Pause and restart support

---

## 🎮 Controls

| Key | Action |
|------|--------|
| ← → | Move |
| Space | Shoot |
| P | Pause |
| Enter | Start / Restart |

---

## 🛠 Built With

- HTML5
- CSS3
- JavaScript
- HTML5 Canvas
- Kiro AI
